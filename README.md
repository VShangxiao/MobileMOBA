# Node.js + Vue.js + MongoDB 开发游戏后台管理系统

## 项目预览





## 01 初始化项目

#### 新建 **server** 文件夹，放服务端项目。



使用 

```
vue create web
```

#### 创建 **web** 项目



使用

```
vue create admin
```

#### 创建 **admin** （后台管理）项目



#### 初始化 **server** 项目

```
npm init -y
```



#### 新建 **index.js** 为入口文件

##### package.json

```
{
  "name": "server",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "serve": "nodemon index.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}
```

##### 安装 nodemon

```
npm -g nodemon
```

##### 

## 02 构建 Element UI 后台管理基础界面

#### 打开 [ELement UI 官网](https://element.eleme.cn/#/zh-CN) ，找到 **组件** ，找到 **Container 布局容器** 

#### 在 admin 文件夹，安装 **element** 插件

```
vue add element
```

#### 继续安装路由

```
vue add router
```

#### 在 admin\src\router.js 文件中引用

##### router.js

```
import Vue from 'vue'
import Router from 'vue-router'
import Home from './views/Home.vue'
import Main from './views/Main.vue'

Vue.use(Router)

export default new Router({
  routes: [
    {
      path: '/',
      name: 'main',
      component: Main
    },
  ]
})
```
